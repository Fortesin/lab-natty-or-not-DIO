# História do primeiro computador e o avanço da tecnologia até os dias atuais.

## 📒 Descrição
Olá nesse projeto criei um texto pelo ChatGPT contando uma breve história da computação e usei o GAMMA para poder gerar alguns slides e imagens para contar essa história.


## 🤖 Tecnologias Utilizadas
Texto = ChatGPT
Slides e imagens = GAMMA

## 🧐 Processo de Criação
Primeiro pedi para o ChatGPT criar uma breve história, segue o comando:
*Escreva um breve texto de 20 linhas , contando a história de como o primeiro computador surgiu e como está a tecnologia nos dias de hoje, como ao longo do tempo a tecnologia se desenvolveu.

Depois pedi para o GAMMA criar um slide e algumas imagens de anime para apresentar essa história.
